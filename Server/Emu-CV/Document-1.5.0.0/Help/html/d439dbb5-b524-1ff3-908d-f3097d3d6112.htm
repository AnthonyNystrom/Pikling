Parameters
contour (IntPtr)
Input contour
pt (PointF)
The point tested against the contour
measureDist (Int32)
If != 0, the function estimates distance from the point to the nearest contour edge
Return Value
When measureDist = 0, the return value is >0 (inside), <0 (outside) and =0 (on edge), respectively. When measureDist != 0, it is a signed distance between the point and the nearest contour edge
