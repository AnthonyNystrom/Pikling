GetSpatialMoment Method (x_order, y_order)

The function GetSpatialMoment retrieves the spatial moment, which in case of image moments is defined as: Mx_order,y_order=sumx,y(I(x,y)?x_order?y_order) where I(x,y) is the intensity of the pixel (x, y).
